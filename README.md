# d3mi.wang

Personal website of Demi Wang — AI researcher at CMU LTI.

Built with Next.js 15, TypeScript, and Tailwind CSS v4. Deployed on Vercel.

## Stack

- **Framework**: Next.js 15 (App Router)
- **Styling**: Tailwind CSS v4 + CSS variables
- **Fonts**: Cormorant Garamond + DM Mono
- **Deployment**: Vercel → [d3mi.wang](https://d3mi.wang)

## Dev

```bash
npm install
npm run dev
```
